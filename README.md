BharatKey IME is an open-source Android Input Method Editor (IME) designed specifically for physical keyboards.

Unlike most Android keyboards, BharatKey IME replicates the Windows Hindi InScript (Devanagari) keyboard layout, English (US) keyboard, and Alt-code character mapping, providing a familiar typing experience for users who rely on external keyboards.

The project was created because Android lacks a keyboard that fully matches the Windows keyboard layout and character mapping required for professional Hindi typing and government typing examinations.

Whether you're practicing on an Android tablet or taking an online typing test with a USB or Bluetooth keyboard, BharatKey IME delivers typing behavior that closely matches Windows.

рџ’Ў Why BharatKey IME?

Android keyboards are primarily designed for touch input and often differ from the Windows Hindi InScript layout.

These differences make it difficult for candidates preparing for government typing examinations where Windows keyboard behavior is the standard.

BharatKey IME bridges this gap by bringing the Windows typing experience to Android devices.
